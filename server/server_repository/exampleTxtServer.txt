● Java фреймворк Netty.

2.Как и в каком виде передавать файлы?
Существует два варианта сериализация и потоком байтов.
3.Как пересылать большие файлы?
Большие файлы удобно пересылать потоком байтов, по кусочкам. Если это сериализация, то файл записывается в один объект, что очень долго и не удобно.
4.Как пересылать служебные команды?
Служебные команды пересылаются через маршрутизатор.
5.Что хранить в базе данных?
В базе данных хранится большое колличество информации, которая не используется постоянно.
6.Как передавать структуру каталогов/файлов?
Это можно сделать на FTP используя маску.
